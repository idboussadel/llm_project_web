SentiTrade-HMA

A Synergistic Framework for Algorithmic Trading via Fusion of Large Language Models and Temporal Fusion Transformers

Overview

SentiTrade-HMA is a production-ready algorithmic trading system that combines context-aware sentiment analysis from fine-tuned Llama-2-7B with Temporal Fusion Transformers for multi-horizon stock price forecasting. The system achieves a Sharpe Ratio of 5.17 and 5% MAE reduction over price-only baselines.

Our platform leverages hierarchical multi-source data aggregation (news articles, social media, and earnings transcripts) to extract financial sentiment, which is then integrated with technical indicators and price data to generate actionable trading signals with confidence intervals and risk metrics.

How It Works

1. Data Collection: The system aggregates financial data from multiple sources including NewsAPI, Finnhub (social media/news), and Financial Modeling Prep (earnings data). Historical price data is fetched via yfinance.

2. Sentiment Analysis: Raw text from each source is processed through our fine-tuned Llama-2-7B model using QLoRA, extracting sentiment scores and confidence levels. The model is specifically trained on financial domain data for superior performance.

3. Hierarchical Aggregation: Sentiment scores from different sources are aggregated using learned weights, accounting for source reliability and temporal relevance. This creates a unified sentiment signal.

4. Feature Engineering: Technical indicators (Hull Moving Average, ATR, RSI, etc.) are calculated and combined with sentiment features and price lags to create a rich feature set for forecasting.

5. TFT Prediction: The Temporal Fusion Transformer model processes the engineered features to generate multi-horizon price predictions with quantile outputs, providing both point forecasts and uncertainty estimates.

6. Signal Generation: Predictions are combined with interpretability insights (attention weights, feature importance) to generate actionable trading signals with risk metrics and confidence intervals.
